# J-VOCA QA 체크리스트

## 테스트 환경

| 환경 | URL |
|---|---|
| 로컬 | `npm run dev -- --host` → `http://<IP>:5173/j-voca/` |
| GitHub Pages | https://ukiband.github.io/j-voca/ |

## 사전 준비

- [ ] 아이폰에서 GitHub Pages URL 접속
- [ ] Safari 공유(↑) → 홈 화면에 추가 → "J-VOCA" 저장
- [ ] Gemini API 키 발급 (https://aistudio.google.com/apikey)
